# nss_data_science_python
NSS Data Science Python exercises

* Learn the basics about Anaconda and Jupyter Notebook
* Learn pandas basics
* Learn the difference between DataFrames and Series
* See how to use .loc, .iloc, and [ ] to subset and filter a DataFrame
* Understand the relationship between base Python, the numpy library, and the pandas library
* Learn about the basic kinds of operations that can be performed on tabular data
* Learn the different types of merges
* Understand the purpose of data visualization
* Distinguish between exploratory and explanatory visuals
* Select an appropriate visual for the data type and goal
* Learn techniques to highlight key insights
* Understand methods of presenting a convincing argument using data
* Python assessment
